# Page fragments

These are fragments of code (html, javascript, php) for inclusion in pages
that are called in the www director.

Basically this is a way to stop those pages getting ridiculously long and confusing.

It is assumed these are rendered within other pages so they may have variable from those pages etc

n.b. reuseable code (object etc) that are likely to be used in multiple places are in the includes/ directory.
